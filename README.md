# student-placement-data
This repository contains a data analysis of student placement outcomes using a Kaggle dataset for 2025. The goal is to explore trends in job placements across different degrees, skills, and scores, providing insights into which areas students are more likely to succeed in, even in the evolving job market influenced by AI.

# Visualizations
Degree vs Placement: Shows the number of students placed vs not placed for each degree.
Placement Percentage by Degree: Shows the percentage of students placed in each degree program.
Internships vs Placement: Visualizes the impact of internship experience on placement outcomes.

# Key Insights
Computer Science and Data Science fields have the highest placement percentages (~66%).
Fields with fewer internships tend to have slightly lower placement rates.
Visual analysis shows that internship experience correlates with better placement outcomes
